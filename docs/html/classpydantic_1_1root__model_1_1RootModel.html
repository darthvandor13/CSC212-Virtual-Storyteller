Detailed Description

!! abstract "Usage Documentation" RootModel and Custom Root Types

A Pydantic BaseModel for the root object of the model.

Attributes: root: The root object of the model. pydantic_root_model: Whether the model is a RootModel. pydantic_private: Private fields in the model. pydantic_extra: Extra fields in the model.

Usage docs: https://docs.pydantic.dev/2.10/concepts/models/#rootmodel-and-custom-root-types

A Pydantic `BaseModel` for the root object of the model.

Attributes:
    root: The root object of the model.
    __pydantic_root_model__: Whether the model is a RootModel.
    __pydantic_private__: Private fields in the model.
    __pydantic_extra__: Extra fields in the model.
