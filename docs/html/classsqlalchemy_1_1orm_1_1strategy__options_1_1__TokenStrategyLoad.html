Detailed Description

Loader strategies against wildcard attributes

e.g.::

    raiseload("*")
    Load(User).lazyload("*")
    defer("*")
    load_only(User.name, User.email)  # will create a defer('*')
    joinedload(User.addresses).raiseload("*")
