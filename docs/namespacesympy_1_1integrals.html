Detailed Description

Integration functions that integrate a SymPy expression.

    Examples
    ========

    >>> from sympy import integrate, sin
    >>> from sympy.abc import x
    >>> integrate(1/x,x)
    log(x)
    >>> integrate(sin(x),x)
    -cos(x)
