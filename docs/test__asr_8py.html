! !

Listens for keywords with NAO’s microphone, sends the recognised word ! to a ChatGPT‑powered webhook, and plays the returned story via ! NAO’s built‑in Text‑to‑Speech (TTS). ! ! This script is intentionally kept as a single, top‑level loop to preserve ! the original project logic. No functions are introduced here so that the ! behaviour remains byte‑for‑byte identical — only comments were added to make ! the file fully Doxygen‑ready. ! ! ### Usage (example) ! ```bash ! python2 test_asr.py # run inside the naoqi_env virtual‑env ! ``` ! !
