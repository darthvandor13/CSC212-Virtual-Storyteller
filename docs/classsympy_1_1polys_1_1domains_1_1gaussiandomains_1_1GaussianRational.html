Detailed Description

Gaussian rational: domain element for :ref:`QQ_I`

    >>> from sympy import QQ_I, QQ
    >>> z = QQ_I(QQ(2, 3), QQ(4, 5))
    >>> z
    (2/3 + 4/5*I)
    >>> type(z)
