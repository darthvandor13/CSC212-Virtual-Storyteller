Detailed Description

The smallest element of a CNF object.

Parameters
==========

lit : Boolean expression

is_Not : bool

Examples
========

>>> from sympy import Q
>>> from sympy.assumptions.cnf import Literal
>>> from sympy.abc import x
>>> Literal(Q.even(x))
Literal(Q.even(x), False)
>>> Literal(~Q.even(x))
Literal(Q.even(x), True)
