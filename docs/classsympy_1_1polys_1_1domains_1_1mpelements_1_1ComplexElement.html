def sympy.polys.domains.mpelements.ComplexElement.parent (   self) 
Get the domain associated with ``self``

Examples
========

>>> from sympy import ZZ, symbols
>>> x, y = symbols('x, y')
>>> K = ZZ[x,y]
>>> p = K(x)**2 + K(y)**2
>>> p
x**2 + y**2
>>> p.parent()
ZZ[x,y]

Notes
=====

This is used by :py:meth:`~.Domain.convert` to identify the domain
associated with a domain element.

Reimplemented from sympy.polys.domains.domainelement.DomainElement.
